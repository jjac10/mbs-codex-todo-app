# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-20

### Added
- Base setup for a Pokédex with Next.js using the Pages Router.
- Centralized Axios instance for consuming PokeAPI.
- Static homepage with the first 151 Pokémon rendered in responsive card grid.
- Dynamic detail route per Pokémon with stats, types, and abilities.
- Setup guide in `docs/setup.md`.
